# Citi Pirates Style Guide - Colors

## Primary Brand Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Pirates Red | `#CC0000` | Primary accent, buttons, highlights |
| Pirates Red Bright | `#FF1A1A` | Hover states, glows |
| Pirates Red Dark | `#990000` | Gradients, darker accents |

## Background Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Pirates Black | `#0A0A0A` | Main background |
| Pirates Black Light | `#1A1A1A` | Elevated surfaces, stat boxes |
| Pirates Black Card | `#141414` | Card backgrounds |

## Neutral Colors

| Color | Hex | Usage |
|-------|-----|-------|
| White | `#FFFFFF` | Primary text |
| Gray | `#888888` | Secondary text, labels |
| Gray Dark | `#333333` | Borders, dividers |

## Accent Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Gold | `#FFD700` | Special highlights (unused currently) |
| Gold Dim | `#B8860B` | Dimmed gold accents |

## Effects

| Effect | Value |
|--------|-------|
| Red Glow | `0 0 20px rgba(204, 0, 0, 0.5)` |
| Gold Glow | `0 0 20px rgba(255, 215, 0, 0.5)` |

## Typography

- **Display Font**: Bebas Neue (headers, jersey numbers)
- **Body Font**: Inter (body text, labels)
